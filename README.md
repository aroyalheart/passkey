# passkey
Secure WHMCS with eHostPK Passkey! 🛡️ Replace passwords with Mac TouchID, FaceID, &amp; Windows Hello. Built with AES-256-GCM encryption &amp; WebAuthn, it offers a 500ms auto-scan login. Features CSRF protection, multi-session logic for Admin/Clients, and sub-folder support. Experience the future of passwordless, biometric security today! 🚀
